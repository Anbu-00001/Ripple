# Branding assets

Designed visual assets for Faultline. The PNGs here are rendered, deterministic
exports — no external fonts or scripts are needed to view them.

| File | Use | Size |
|------|-----|------|
| `faultline-readme-banner.png` | Top of the repo `README.md` | wide banner |
| `faultline-hero.png` | Devpost hero image · video establishing card | 1600×900 |
| `faultline-og-card.png` | Devpost gallery / social (OG) preview | 1200×630 |

## Design language
- **Palette (Okabe–Ito / Wong, colorblind-safe):** `#56B4E9` changed · `#E69F00`
  untested · `#009E73` impacted-tested · `#D55E00` gate/blocked · bg `#0B1220`.
- **Shape redundancy (legible without color):** changed = circle · untested =
  diamond · impacted = square/rect. The same language is used by the inline
  Mermaid verdict graph and the interactive CI artifact, so every Faultline
  visual reads the same way.

The hero and OG card carry the project URL in the footer; the README banner does
not. Source designs live in the Claude Design project for this repo.
